##Syndetics

يمكن إضافة بيانات اعتماد Syndetics عبر بيانات الطرف الثالث الإثرائية في قائمة الإدارة. يمكن أن تفيد Syndetics في إضافة محتوى إثرائي إلى أسبن بما في ذلك الأغلفة والمعلومات الأخرى.
